## 2.3 常用格式化符号
* %v：默认格式，打印任何数据类型。

* %s：打印字符串。

* %d：打印十进制整数。

* %f：打印浮点数。

* %t：打印布尔值。

* %p：打印指针地址。

* %+v：打印结构体时，带字段名。

# 3.小练习
* 输出格式化：用fmt.Printf打印一个浮点数、一个字符串、一个整数
* 格式化输入：用fmt.Scan 从标准输入读取一个整数和一个字符串
* 字符串拼接： 用fmt.Sprintf 拼接多个变量为一个字符串并输出
